Sniper Server Application

Welcome to the On-Chain Dynamics Sniper Scannerâ€”an advanced, multi-chain decentralized finance (DeFi) tool designed for professional traders and developers. This platform enables real-time liquidity scanning, transaction automation, and configuration management across multiple blockchains.

Overview

The Sniper Scanner Application is a comprehensive tool for identifying liquidity pool events, automating transactions, and managing blockchain-specific configurations. With support for blockchains such as Ethereum, Solana, and Tron, the application offers unparalleled flexibility and efficiency for DeFi traders and developers.

Key Features

  • Multi-Chain Compatibility: Supports Ethereum, Solana, Tron, and more.
  • Real-Time Liquidity Scanning: Monitors liquidity pool events on Uniswap V2, Raydium, and SunPump.
  • Transaction Automation: Executes transactions based on user-defined configurations.
  • Robust Configuration Management: Manages dynamic configurations for liquidity thresholds, token prices, and more.
  • Scalable Architecture: Utilizes worker threads and rate-limited queues for efficient performance.
  • Error Handling and Resilience: Implements retry mechanisms and fallback data for robust operation.

Modules

1. Clients

  • DynamoDBClient: Manages DynamoDB client initialization, retries, and resource cleanup.
  • JupiterClient: Interfaces with the Jupiter API for Solana, supporting liquidity scanning and token metadata retrieval.
  • MongoDBClient: Handles MongoDB initialization, retries, and database operations.

2. Connections

  • EthereumWeb: Manages Ethereum RPC and WebSocket connections for transaction and data retrieval.
  • SolanaWeb: Handles Solana RPC and WebSocket connections with health monitoring and retry logic.
  • TronWeb: Provides Tron RPC connection utilities with robust health checks and reconnection logic.

3. Scanners

  • EthereumGasMonitor: Tracks Ethereum gas prices and block data in real-time.
  • EthereumMempoolScanner: Monitors Ethereum mempool for Uniswap V2 transactions.
  • SolanaLogScanner: Scans Solana logs for Raydium liquidity pool events.
  • TronEventScanner: Tracks Tron SunPump contract events with automatic retries.

4. Blockchain Utilities

  • EthereumTransactions: Processes Ethereum transactions, including decoding, monitoring, and executing trades.
  • SolanaTokenLiquidity: Manages Solana token liquidity data, metadata, and price information.
  • TronTokenLiquidity: Retrieves liquidity and metadata for tokens on the Tron blockchain.

5. Redis Utilities

  • RedisChannels: Manages Redis channels for multi-chain configurations and updates.
  • RedisSubscriptions: Handles dynamic subscriptions and real-time Redis message processing.
  • RedisSync: Periodically synchronizes in-memory configurations with Redis.

6. Sniper Functions

  • SniperHandlers: Manages real-time updates to sniper configurations.
  • SniperFunctions: Provides utilities for managing sniper configurations, including retrieval, updates, and state changes.

7. Raydium Utilities

  • RaydiumUtilities: Handles liquidity pool transactions and events for Raydium on the Solana blockchain.

8. Worker Management

  • WorkerManagement: Monitors and manages worker threads for blockchain scanning and asynchronous tasks.
  • ScannerRestartUtility: Implements retry mechanisms with exponential backoff for scanner processes.

9. Utility Modules

  • UtilityFunctions: Provides logging, retry logic, terminal customization, and notification tools.
  • ApplicationBootstrap: Manages application initialization, service monitoring, and graceful shutdown.

Use Cases

  1. Liquidity Sniping: Identify and act on new liquidity pool events in real-time.
  2. Arbitrage Opportunities: Monitor price discrepancies across DEXs for profitable trades.
  3. Token Launch Participation: Automate transactions during high-demand token launches.
  4. Custom Trading Strategies: Implement personalized strategies with dynamic configurations.

Technology Stack

  • Node.js & Express: Backend framework for APIs and WebSocket management.
  • Blockchain SDKs: Ethers.js, Web3.js, TronWeb, and Jupiter API for blockchain interactions.
  • Redis: Real-time messaging and configuration management.
  • MongoDB & DynamoDB: Databases for configuration and transaction storage.
  • Worker Threads: Asynchronous task execution and blockchain scanning.
