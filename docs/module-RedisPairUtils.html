RedisPairUtils

Provides utility functions for formatting numbers, handling intervals, and sanitizing data.

This module includes the following utilities:

  • formatRegularNumber: Formats a number into a string with suffixes (e.g., K, M, B, T).
  • formatDecimalNumber: Formats a decimal number with currency symbols and suffixes.
  • formatNumber: Formats a BigNumber into a human-readable string with suffixes.
  • getIntervalStartTime: Calculates the start time of a specified interval (e.g., 5m, 1Hr).
  • getTransactionIntervalStartTime: Calculates the start time of a transaction interval (e.g., 5m, 15m, 30m, 1hr, 24hr).
  • sanitizeData: Sanitizes an object by handling null/undefined values and converting objects/arrays to JSON strings.
  • safeParseFloat: Safely parses a value into a float, defaulting to 0 if the value is NaN.
