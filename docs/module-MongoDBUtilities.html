MongoDBUtilities

Provides utility functions for interacting with MongoDB to manage user wallets and save transactions.

This module includes:

  • getUserWallets: Fetches user wallets from the ocdWallets collection in MongoDB.
  • getUserWalletsAndKeys: Retrieves user wallets and their corresponding private keys based on selected wallet addresses.
  • saveSniperTransaction: Saves sniper transaction records in the ocdTransactions collection in MongoDB.

These utilities facilitate efficient retrieval, filtering, and storage of user and transaction data in a MongoDB database.
