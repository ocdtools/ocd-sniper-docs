ReferralCodeIndex

Global Secondary Index (GSI) for the ocdReferrals table. This index enables efficient querying and grouping of referral-related data by referral code.

Index Schema:

  • Partition Key: referralcode (Referral code used to group referral data).
  • Projection Type: ALL (Includes all attributes in the projection).

Key Features:

  • Referral-Based Grouping: Enables grouping and querying of data based on referral codes.
  • Comprehensive Projection: Includes all attributes for detailed analysis of referral performance.

Use Cases:

  • Analyzing referral program performance by specific referral codes.
  • Tracking trade volumes, values, and counts linked to each referral code.
  • Rewarding users based on the performance of their referral codes.
