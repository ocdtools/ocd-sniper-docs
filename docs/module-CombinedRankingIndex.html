CombinedRankingIndex

Global Secondary Index (GSI) for the ocdUserStats table. This index is designed for leaderboard ranking based on total points.

Index Schema:

  • Partition Key: totalpoints (Total points used to rank users on the leaderboard).
  • Projection Type: ALL (Includes all attributes in the projection).

Key Features:

  • Leaderboard Ranking: Facilitates sorting and ranking of users based on their total points.
  • Comprehensive Projection: Includes all attributes for detailed leaderboard analytics.

Use Cases:

  • Generating real-time leaderboard rankings.
  • Analyzing user activity and engagement based on total points.
  • Supporting gamification and incentive programs.
