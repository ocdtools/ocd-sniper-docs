GeckoConfig

Provides configuration settings for interacting with the CoinGecko API.

  • Validates the presence of required environment variables on initialization.
  • Exports a structured configuration object for CoinGecko API operations.

Members

(static, constant) GECKO_CONFIG :Object

Description:

  • CoinGecko API configuration settings derived from environment variables.

    • GECKO_PRO_API: API key for CoinGecko Pro API access.
    • GECKO_BASE_URL: Base URL for the CoinGecko API.
    • MAX_RETRIES: Maximum retry attempts for API requests.
    • RETRY_DELAY_MS: Delay (in milliseconds) between retry attempts.
    • MAX_BACKOFF_MS: Maximum delay (in milliseconds) for exponential backoff during retries.
Properties:
Name Type Description
GECKO_PRO_API string

API key for accessing the CoinGecko Pro API.
GECKO_BASE_URL string

Base URL of the CoinGecko API.
MAX_RETRIES number

Maximum number of retry attempts for API requests.
RETRY_DELAY_MS number

Delay between retries in milliseconds.
MAX_BACKOFF_MS number

Maximum backoff time in milliseconds for retries.
