SniperTransactionIndex

Global Secondary Index (GSI) for the ocdTransactions table. This index enables efficient querying of transactions based on user ID and transaction type.

Index Schema:

  • Partition Key: userid (User ID used to group transactions).
  • Sort Key: txtype (Transaction type for sorting, e.g., "sniper", "platform").
  • Projection Type: ALL (Includes all attributes in the projection).

Key Features:

  • User Grouping: Allows transactions to be grouped and queried by user ID.
  • Transaction Sorting: Enables sorting and filtering by transaction type.
  • Comprehensive Projection: Includes all attributes for flexible querying.

Use Cases:

  • Retrieving all transactions for a specific user.
  • Filtering transactions by type, such as sniper or platform trades.
  • Analyzing user-specific transaction patterns.
