ocdTotalStats

Consolidates platform-wide metrics for analyzing and reporting overall performance.

Table Schema:

  • Primary Key: statsid: Static value, 'ALL_USERS'.
  • Attributes:
    • totaltradevolume: Aggregated trade volume across all users.
    • totalvalue: Total value of trades in ETH.
    • totalvalueusd: Total value of trades in USD.
    • totaltradecount: Total number of trades.
    • totalusers: Total number of registered users.
    • totalreferrals: Total number of referral-based signups.
    • totaluserpoints: Cumulative leaderboard points of all users.
    • totalreferralpoints: Points earned via referrals.

Key Features:

  • Platform Analytics: Tracks global trade and user metrics.
  • Referral Program Insights: Monitors referral-based activity and impact.
  • Real-Time Updates: Dynamically updates global statistics via DynamoDB Streams.

Use Cases:

  • Generating platform-wide performance dashboards.
  • Supporting decision-making with aggregated data.
  • Monitoring growth trends and key performance indicators (KPIs).

This module represents the ocdTotalStats table schema and provides aggregated data for all users and trades on the platform.

Table Definitions

ocdTotalStats

Description:

  • The ocdTotalStats table stores aggregated statistics for all users of the platform. It provides a consolidated view of the overall performance and activity across the dApp, enabling insights into global usage metrics and referral impact.
